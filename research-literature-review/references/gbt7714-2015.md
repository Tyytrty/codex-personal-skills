# GB/T 7714—2015 顺序编码制执行规则

本说明依据用户提供的《GB/T 7714—2015 信息与文献 参考文献著录规则》整理，用于本 skill 的 Markdown 综述。标准允许顺序编码制和著者—出版年制；本 skill 固定采用顺序编码制，以满足“按被引先后顺序编号”的要求。

## 文内标注与排序

- 文献在正文第一次出现时获得连续序号，序号置于方括号内，如 `[1]`。
- 同一处引用多篇文献时全部列入同一组，如 `[2, 5, 6]`；连续序号可在人工排版阶段写成 `[2-4]`，自动流程默认保留逐项编号以避免歧义。
- 同一文献再次被引时沿用原序号，不生成新条目。
- 参考文献表按正文首次被引的序号排列，条目前置 `[1]`、`[2]`，不得使用 `1.`、`2.`。

## 责任者与文字

- 责任者不超过 3 人时全部著录；超过 3 人时著录前 3 人，中文条目后加“等”，西文条目后加 `et al.`。
- 西文个人责任者采用姓在前、名缩写在后的形式；中文姓名按原文著录。生成前应在 BibTeX 中提供已经规范化的 `author` 字段。
- 原则上使用信息资源本身的语种著录，不擅自翻译题名。西文责任者的字母大小写应符合来源文种习惯。

## 文献类型与载体标识

- 普通图书 `M`，期刊文章 `J`，会议录/会议文献 `C`，学位论文 `D`，报告 `R`，标准 `S`，专利 `P`，报纸文章 `N`，汇编 `G`，档案 `A`，舆图 `CM`，数据集 `DS`，数据库 `DB`，计算机程序 `CP`，电子公告 `EB`，其他 `Z`。
- 在线电子资源在类型后附 `/OL`，如 `[J/OL]`、`[D/OL]`、`[EB/OL]`。

## 常用著录格式

```text
[序号] 主要责任者. 题名[J]. 刊名, 年, 卷(期): 起止页码. DOI:标识符.
[序号] 主要责任者. 题名[J/OL]. 刊名, 年, 卷(期): 起止页码[引用日期]. URL. DOI:标识符.
[序号] 主要责任者. 书名[M]. 版本. 出版地: 出版者, 出版年: 引文页码.
[序号] 析出文献责任者. 析出题名[M]//专著责任者. 专著题名. 出版地: 出版者, 出版年: 析出页码.
[序号] 主要责任者. 题名[C]. 出版地: 出版者, 出版年.
[序号] 主要责任者. 题名[D]. 保存地: 保存单位, 年.
[序号] 主要责任者. 题名[R]. 出版地: 出版者, 年.
[序号] 标准责任者. 标准名称: 标准号[S]. 出版地: 出版者, 出版年.
[序号] 专利申请者或所有者. 专利题名: 专利号[P]. 公告或公开日期.
[序号] 主要责任者. 题名[EB/OL]. (更新或修改日期)[引用日期]. URL.
```

## 电子资源和 DOI

- 在线资源应记录实际获取和访问路径，并在可获得时记录引用日期，格式为 `[YYYY-MM-DD]`。
- DOI 是数字对象唯一标识符；若来源同时给出 URL 与 DOI，可二者均著录。DOI 写作 `DOI:10.xxxx/...`。
- 纯 DOI 链接不应伪造引用日期；缺少 `urldate` 时校验器给出警告，要求在交付前人工核实。

## 标点与缺项

- 题名、文献类型标识之后用句点；出版地与出版者之间用冒号；出版者与年份、期刊名与年份、年份与卷号之间用逗号；卷期后的页码前用冒号。
- 每条参考文献以句点结束。
- 第一版通常不著录版本项；其他版本应著录。
- 无出版地的中文文献可写 `[出版地不详]`，西文可写 `[S.l.]`；无出版者的中文文献可写 `[出版者不详]`，西文可写 `[s.n.]`。电子资源在确无该项时可省略，不能凭空补造。

## BibTeX 字段约定

- `author`, `title`, `year` 为核心字段；期刊文章还应有 `journal`，并尽量有 `volume`, `number`, `pages`。
- 图书、会议录、报告、标准尽量提供 `address`, `publisher`, `edition`, `pages`。
- 学位论文提供 `school` 和 `address`；专利提供 `number` 或 `patentnumber` 及 `date`。
- 在线资源提供 `url` 与 `urldate`；DOI 提供 `doi`。
- 特殊类型可用 `typecode={DS}`、`typecode={EB}` 等覆盖自动映射。

## 最终人工核查

自动脚本负责定序、基本类型映射和标点拼接，但不能从残缺元数据中恢复真实的出版地、出版者、引用日期或责任者拼写。交付前必须逐条对照 DOI 页面、出版者页面或文献原文，修正 BibTeX 后重新生成并校验。
