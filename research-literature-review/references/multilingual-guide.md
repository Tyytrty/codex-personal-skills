# 多语言 Markdown 综述指南

## 原则

- 先完成并验证源语言 `{topic}_review.md`，再翻译。
- 保留数字引用的位置和编号，不重新排序参考文献。
- 保留 Markdown 标题、列表、表格、公式、链接和图片相对路径。
- 图注中的作者、原文图号、DOI 和许可证不得翻译成不准确形式。
- 参考文献题名默认保留原文；用户明确要求时才翻译题名，并同时保留原题名。

## 流程

1. 复制已经通过 `validate_review_md.py` 的 Markdown。
2. 只翻译正文和说明性图注，不改 `[1]`、`[2, 3]` 等引用标记。
3. 不改 `![alt](relative/path.png)` 中的路径，可翻译 alt 文本。
4. 再次运行 `validate_review_md.py`，确认引用顺序、参考文献和图片来源仍一致。
5. 默认交付翻译后的 Markdown；不生成 Word。用户明确要求时才额外导出 PDF。

## 语言注意

- 中文、日文等不以空格分词的语言按字符核算正文长度。
- 德语、法语、西班牙语和英语按词核算，并允许因翻译产生合理长度变化。
- 缩略词首次出现时展开；公认缩略词可以保留。
