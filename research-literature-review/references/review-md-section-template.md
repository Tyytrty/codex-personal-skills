# Markdown 综述结构、引用与原文图模板

## 正文骨架

```markdown
# 题目

## 摘要

单段：背景、核心认识、主要争议、研究方向。

## 1 引言

问题重要性、已有共识、知识缺口和全文中心论点 [@key]。

## 2 主体主题一

主张、证据、解释、边界 [@key1; @key2]。

## 3 主体主题二

主张、证据、解释、边界 [@key3]。

## 4 讨论

异质性、矛盾、失败原因和外推边界。

## 5 展望

可检验研究议程与工程约束。

## 6 结论

回收中心论点，不引入新证据。
```

## 引用规则

- 草稿阶段只使用 `[@bibkey]` 或 `[@key1; @key2]`。
- 引用紧跟被支撑的句子，放在句号之前或之后时全文保持一致。
- 定稿脚本按首次出现顺序生成 `[1]`、`[2]`、`[3]`。
- 同一文献再次出现时沿用原编号。
- 多篇并列写作 `[2, 4]`；不要因编号不连续而改动证据顺序。
- 参考文献表按编号排列，不按作者、年份或评分排序。
- 参考文献表按 GB/T 7714—2015 顺序编码制著录：条目以 `[1]` 起始，包含文献类型/载体标识，具体规则见 `gbt7714-2015.md`。

常用格式示例：

```text
[1] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码. DOI:10.xxxx/xxxx.
[2] 作者. 题名[J/OL]. 刊名, 年, 卷(期): 起止页码[YYYY-MM-DD]. URL. DOI:10.xxxx/xxxx.
```

## 原文图模板

```markdown
![螺旋光纤对不同入射角波场的方向响应]({主题}_review_assets/duan2025_fig10.png)

图 2. 缠绕角与入射角共同改变 P 波和 S 波的方向响应。来源：Duan et al. (2025)，原文图 10，DOI：[10.xxxx/xxxxx](https://doi.org/10.xxxx/xxxxx)，CC BY 4.0。
```

图后正文必须回答：

1. 图中哪一关系支撑本节主张。
2. 该图对应什么实验、模拟或边界条件。
3. 该图不能被外推到哪些场景。

## 图片筛选优先级

1. 机制图或方向响应图。
2. 能同时比较关键方案的实验构型或结果图。
3. 汇总多个变量关系的等值图、相图或多面板图。
4. 具有代表性的现场验证图。

不选择期刊封面、装饰图、普通设备照片、与正文主张重复的多幅近似图。

## 最终检查

- [ ] Markdown 是唯一正式正文。
- [ ] 不存在 `.docx` 交付物。
- [ ] 首次出现的新引用编号从 1 连续递增。
- [ ] 参考文献表与首次被引顺序一致。
- [ ] 参考文献采用 GB/T 7714—2015 方括号序号、类型/载体标识及对应著录格式。
- [ ] 所有参考文献均在正文被引。
- [ ] 图片为相对路径且文件可打开。
- [ ] 每幅图均有原文图号、作者、DOI/链接和许可证。
- [ ] 图后有分析，而非只放图和图注。
